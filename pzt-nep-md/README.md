# NEP Potential for Pb(Zr,Ti)O3 (PZT)

This repository contains a Neuroevolution Potential (NEP) for Pb(Zr1-xTix)O3 (PZT), together with training datasets and molecular dynamics (MD) examples.

The model is designed for large-scale atomistic simulations of phase evolution, octahedral rotation dynamics, and electromechanical response in PZT across compositions (x=0.0-1.0) and temperatures (10-1000 K).

---

# 📁 Repository Structure

- `nep_model/` : Final NEP model, hyperparameters, and utilities  
- `training_data/` : Dataset used for training  
- `md_examples/` : Example MD simulations for physical applications
